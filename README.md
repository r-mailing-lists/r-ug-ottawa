# r-ug-ottawa

Ottawa R User Group

Automated mbox archive. Updated daily via GitHub Actions.

## Structure

- `raw/` - Monthly .mbox files downloaded from [https://stat.ethz.ch/pipermail/r-ug-ottawa/](https://stat.ethz.ch/pipermail/r-ug-ottawa/)
- `processed/` - Structured JSON output from rmail-parser
- `meta.json` - List statistics
